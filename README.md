# 📊 E-commerce Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Power BI dashboard** developed using an E-commerce Sales dataset to analyze business performance and generate actionable insights.

The dashboard enables users to explore sales trends, customer purchasing behavior, product performance, delivery efficiency, and payment preferences through interactive visualizations, slicers, and custom tooltips.

---

## 🎯 Project Objectives

- Analyze overall sales performance
- Identify top-performing products and categories
- Monitor monthly profit trends
- Compare state-wise sales
- Analyze payment mode preferences
- Evaluate product delivery performance
- Build an interactive dashboard for business decision-making

## 📌 Business Insights

- Clothing category generated the highest revenue.
- Cash on Delivery (COD) was the most preferred payment mode.
- Monthly profits showed seasonal fluctuations.
- Certain products required longer delivery times than others.
- Interactive filters allow quick comparison across states and quarters.
